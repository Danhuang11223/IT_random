# AI Use Statement Template

Use this appendix wording only if it is honest. Do not claim "no GenAI use" if AI influenced the code, tests, debugging, or report.

## Recommended Declaration

Declaration on the use of Generative AI:

- We declare that we have used GenAI for debugging support (interpreting error messages and suggesting possible fixes), and we implemented and tested the final solution ourselves.
- We declare that we have used GenAI to generate or refine small, self-contained code snippets (e.g., helper functions/validation), and we adapted them to our project and take responsibility for the final code.
- We declare that we have used GenAI for refactoring and code-quality suggestions (e.g., reducing duplication, improving structure), and we implemented the changes ourselves.
- We declare that we have used GenAI to suggest unit test ideas and edge cases, and we wrote/validated the final tests ourselves.
- We declare that we have used GenAI to support accessibility implementation (e.g., labels, focus handling, keyboard navigation) and validated the outcome ourselves.
- We declare that we have used GenAI to support sustainability/performance improvements (e.g., interpreting optimisation suggestions), and we verified the changes using tooling and manual checks.
