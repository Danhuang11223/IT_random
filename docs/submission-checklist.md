# Submission Checklist

Use this as the final hand-in checklist.

## Required files

- [ ] Export the written report to PDF
- [ ] Zip the full application code
- [ ] Confirm the public repository link works
- [ ] Confirm the deployed application URL works (`https://it-random.onrender.com`)

## Report content

- [ ] Introduction with repository link and deployed URL
- [ ] Updated design specification (user stories, architecture, ER, sitemap, wireframes)
- [ ] Implementation highlights
- [ ] Testing section with evidence
- [ ] Accessibility section with at least 3 implemented improvements
- [ ] Sustainability section with before/after measurements
- [ ] Appendix with team contributions
- [ ] Appendix with AI Use Statement

## Technical verification

- [ ] Run `.venv/bin/python manage.py test daily_random_events`
- [ ] Run `.venv/bin/python manage.py check`
- [ ] Run `cd frontend && npm run build`
- [ ] Check the deployed application manually on desktop and mobile widths
